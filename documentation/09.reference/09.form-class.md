Create object
-------------

Create a object of the `Html\Form` class:

```php
$form = new Html\Form(Registry $registry, Language $language);
```


Init
----

Init the class:

```php
$form->init(array $attributeArray, array $optionArray) : self;
```


Legend
------

Append the legend:

```php
$form->legend(string $html, array $attributeArray) : self;
```


----
