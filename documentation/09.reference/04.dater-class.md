Create object
-------------

Utilize the constructor of the `Dater` class:

```php
$dater = new Dater();
```


Init
----

Init the class:

```php
$dater->init(int $date) : void
```


Get time zone
-------------

get the time zone:

```php
$dater->getTimeZone() : DateTimeZone
```


Get date time
-------------

get the date time:

```php
$dater->getDateTime() : DateTime
```


Format time
-----------

format to time:

```php
$dater->formatTime() : string
```


Format date
-----------

format to date:

```php
$dater->formatDate() : string
```


Format field
------------

format to field:

```php
$dater->formatField() : string
```
