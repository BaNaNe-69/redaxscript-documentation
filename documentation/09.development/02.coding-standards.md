General
-------

We prefer the Allman style and tabs to define blocks of code:

```
if ($hello === 'world')
{
	$output = 'hello';
}
else
{
	$output = 'world';
}
```


Styles
------

We prefer [NCSS](https://github.com/redaxmedia/ncss) to give our `PostCSS` a semantic meaning:

```
.rs-text-hello
{
	&.rs-is-active
	{
		display: block;
	}
}
```
