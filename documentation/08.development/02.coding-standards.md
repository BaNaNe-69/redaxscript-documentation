General
-------

We prefer the Allman style and tabs to indent code:

```php
if ($condition)
{
	return 'hello';
}
else
{
	return 'world';
}
```


Styles
------

We prefer [NCSS](https://ncss.io) to give our `PostCSS` a semantic meaning:

```css
.rs-link-default
{
	color: gray(0)
	text-decoration: underline;

	&:hover
	{
		color: var(--rs-color-primary);
	}
}
```
