Get registry
------------

Get item from the registry:

```php
<?php Helper::getRegistry(string $key); ?>
```


Get language
------------

Get item from the language:

```php
<?php Helper::getLanguage(string $key); ?>
```


Get setting
-----------

Get item from the settings:

```php
<?php Helper::getSetting(string $key); ?>
```


Get title
---------

```php
<?php Helper::getTitle(); ?>
```
