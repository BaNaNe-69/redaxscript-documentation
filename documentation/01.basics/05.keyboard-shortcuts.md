Form
----

Unmask password field:

```
CLTR + ALT + U
```

Trigger okay action:

```
CLTR + ALT + O
```

Trigger cancel action:

```
CLTR + ALT + C
```

Trigger submit action:

```
CLTR + ALT + S
```


Misc
----

Jump to login and logout:

```
CLTR + ALT + L
```

Toggle admin docks:

```
CLTR + ALT + D
```

Toggle admin panel:

```
CLTR + ALT + P
```
