Form
----

Unmask the active password field:

```
CLTR + ALT + U
```

Trigger the okay action:

```
CLTR + ALT + O
```

Trigger the cancel action:

```
CLTR + ALT + C
```

Trigger the submit action:

```
CLTR + ALT + S
```


Misc
----

Jump to the login and logout:

```
CLTR + ALT + L
```

Toggle the admin docks:

```
CLTR + ALT + D
```

Toggle the admin panel:

```
CLTR + ALT + P
```
